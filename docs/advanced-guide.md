# 进阶使用指南

## 图片管理最佳实践

### 目录结构

建议的目录结构：

```plaintext
my-image-bed/
├── img/
│   ├── blog/          # 博客文章图片
│   ├── avatar/        # 头像类图片
│   └── screenshots/   # 截图类图片
└── README.md
```

合理的目录结构可以帮助你更好地管理图片资源。

## 分支管理建议

- 始终使用 master 分支存储图片
- 可以使用其他分支进行测试，但不要用于生产环境
- 不要删除 master 分支，这会导致 CDN 链接失效

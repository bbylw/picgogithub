# PicGo-GitHub 图床搭建指南

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

这是一个帮助你使用 GitHub 和 PicGo 搭建免费图床的教程，支持 CDN 加速，适用于个人网站、博客或其他平台。

## 🚀 特性

- 免费无限空间
- CDN 加速支持
- 简单易用的图形界面
- 支持拖拽上传
- 自动生成图片链接

> **重要提示**：本项目必须使用 master 分支以确保 CDN 加速正常工作。
